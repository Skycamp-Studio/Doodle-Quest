Doodle Quest

By SkyCamp Studio

About the Game

Doodle Quest revolutionizes the classic “Draw a line” mobile game by bringing it into a 3D realm. This innovative mixed reality game offers intellectually stimulating “Draw-to-Win” puzzle challenges, seamlessly blending the familiar drawing mechanic with immersive 3D environments. The game offers a refreshingly new and engagingly challenging experience.

Features

  • Act as a mischievous magic artist using a magic brush to create physics-based magical touches to solve seemingly impossible challenges.
  • Enjoy over 200 exquisitely designed levels, 20+ distinctive composition modules, and 3 types of drawing materials with different physics properties.
  • Immerse yourself in an effortlessly learnable and delightfully playable experience with a charming art style and soothing music.
  • Supports MR see-through and hand-tracking game controls for a truly immersive experience.
  • (COMING SOON) Use the level editor to create countless possibilities for community-created levels.
